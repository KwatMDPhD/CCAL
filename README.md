# See big picture

CCAL itself doesn't have much code. Instead, CCAL uses other cool libraries as submodules in `ccal/`, and import their awesome functions inside `ccal/__init__.py`'.

# Install

```sh
pip install ccal
conda install -c r rpy2 r-mass
```
